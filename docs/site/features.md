# Features

## Fast, Trustworthy Analytics
- Upload CSVs and get instant, data-backed answers.
- Natural language chat with citations and summaries.
- Export insights, charts, and chat history.

## Modern Tech Stack
- Next.js 16, React 19, Tailwind CSS v4, shadcn/ui.
- Drizzle ORM, Neon Postgres, Upstash Redis.
- Vercel AI SDK with multi-provider support.

## Secure & Scalable
- Multi-tenant authentication (better-auth).
- Edge-protected endpoints (proxy.ts).
- Rate limiting and caching.

## Beautiful, Exportable Charts
- Bar, line, pie, scatter charts.
- Export as PNG, SVG, Markdown, TXT.

## RAG & Multi-Provider AI
- Retrieval-augmented generation for large datasets.
- Automatic provider selection for best cost/quality.

## Production-Ready
- Typed, composable modules.
- Strict TypeScript, robust test coverage.
- Clean dashboard UX, billing, and usage tracking.
